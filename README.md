# restaurant_dashboard
Dự án Restaurant Dashboard là một công cụ quản lý và phân tích dữ liệu chuyên biệt dành cho các nhà hàng Mục tiêu chính là cung cấp một cái nhìn tổng quan, trực quan và theo thời gian thực về hiệu suất kinh doanh của nhà hàng. Dashboard này giúp các nhà quản lý đưa ra quyết định dựa trên dữ liệu bằng cách hiển thị các chỉ số hiệu suất chính (KPIs).
